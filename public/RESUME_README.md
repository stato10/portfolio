# Resume PDF Setup

## Instructions

1. **Create or export your resume as a PDF**
2. **Name it exactly**: `resume.pdf`
3. **Place it in**: `public/resume.pdf`
4. The download button will automatically work once the file is in place

## Recommended Resume Content

For a Full Stack Engineer position, include:
- Contact Information
- Professional Summary
- Technical Skills (matching your portfolio)
- Work Experience
- Projects (link to your portfolio)
- Education
- Certifications (if any)

## File Size

- Keep the PDF under 2MB for fast loading
- Optimize images if included
- Use standard PDF format (PDF/A recommended)
